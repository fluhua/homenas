FamilyNAS 客户端隐私政策

FamilyNAS 是自托管私有云相册与备份工具。客户端用于把照片、视频和家庭重要时刻备份到用户自己的 NAS 或家庭服务器，并通过共享相册与家人协作。

中文版

本政策适用于 FamilyNAS iOS 与 Android 客户端。FamilyNAS 不提供公共云存储服务；照片、视频和文件默认存储在用户自己安装并管理的 FamilyNAS 服务端上。

1. 我们访问的数据

数据类型用途是否离开设备
照片、视频、实况照片用户选择备份、浏览、预览或加入共享相册时使用仅传输到用户自己的 FamilyNAS 服务端
设备名称、系统版本、App 版本在服务端识别设备，便于多设备管理仅传输到用户自己的 FamilyNAS 服务端
局域网信息自动发现同一 Wi-Fi 下的 FamilyNAS 服务端不上传到我们的服务器
相机扫描服务端二维码以快速连接不上传到我们的服务器
崩溃日志、诊断信息、基础使用事件通过 Firebase Analytics 和 Crashlytics 改进稳定性和体验可能发送至 Google/Firebase

2. 数据存储位置

用户的照片、视频、共享相册内容和文件元数据存储在用户自己的 NAS、Mac、Windows 电脑或其他家庭服务器上。FamilyNAS 客户端不会把用户的媒体文件上传到我们的公共服务器。

3. 分享与共享相册

FamilyNAS 支持家庭成员通过共享相册记录和分享孩子成长经历、旅行、生日、节日聚会和其他家庭重要时刻。共享内容只会在用户自己的服务端和被授权的家庭成员之间访问。

4. 安全措施

  • 客户端与服务端通过 HTTPS/TLS 加密通信。
  • 客户端使用证书指纹校验，降低中间人攻击风险。
  • 家庭成员使用独立账号，文件默认隔离，仅用户主动共享的内容才会出现在共享相册中。

5. 我们不做什么

  • 不出售、出租或交易用户个人信息。
  • 不使用广告 SDK。
  • 不追踪用户跨 App 或跨网站行为。
  • 不在我们的服务器保存用户照片、视频或家庭相册内容。

6. 用户控制权

用户可以在系统设置中撤销相册、相机、局域网、通知等权限；也可以在 FamilyNAS 服务端管理页面删除文件、停用设备或删除账号。卸载客户端会清除本机缓存和登录令牌。

7. 儿童隐私

FamilyNAS 可用于家庭成员保存和分享孩子成长记录，但我们不会主动收集儿童个人信息。相关媒体文件由用户自行保存在自己的私有服务端。

8. 联系我们

如有隐私相关问题，请通过 huasen520520@outlook.com 联系我们。

English Version

This policy applies to the FamilyNAS iOS and Android clients. FamilyNAS does not provide public cloud storage. Photos, videos, and files are stored on the FamilyNAS server installed and managed by the user.

1. Data We Access

DataPurposeLeaves the device?
Photos, videos, and Live PhotosBackup, browsing, previewing, and shared albums selected by the userOnly transferred to the user's own FamilyNAS server
Device name, OS version, app versionIdentify and manage devices on the user's serverOnly transferred to the user's own FamilyNAS server
Local network informationDiscover a FamilyNAS server on the same Wi-Fi networkNot uploaded to our servers
CameraScan the server QR code for quick connectionNot uploaded to our servers
Crash logs, diagnostics, and basic usage eventsImprove stability and app experience through Firebase Analytics and CrashlyticsMay be sent to Google/Firebase

2. Where Data Is Stored

The user's photos, videos, shared album content, and file metadata are stored on the user's own NAS, Mac, Windows PC, or other home server. The FamilyNAS client does not upload user media files to our public servers.

3. Sharing and Shared Albums

FamilyNAS helps family members record and share a child's growth, trips, birthdays, holidays, and other important family moments through shared albums. Shared content is accessible only through the user's own server and authorized family members.

4. Security

  • The client communicates with the server over HTTPS/TLS.
  • Certificate fingerprint verification helps reduce man-in-the-middle risks.
  • Family members use separate accounts. Files are private by default, and only content selected by the user appears in shared albums.

5. What We Do Not Do

  • We do not sell, rent, or trade personal information.
  • We do not use advertising SDKs.
  • We do not track users across apps or websites.
  • We do not store user photos, videos, or family album content on our servers.

6. User Control

Users can revoke photo library, camera, local network, and notification permissions in system settings. Users can also delete files, revoke devices, or remove accounts from the FamilyNAS server admin page. Uninstalling the client clears local caches and login tokens.

7. Children's Privacy

FamilyNAS may be used by families to store and share a child's growth records, but we do not knowingly collect children's personal information. Related media files are stored by users on their own private servers.

8. Contact

For privacy questions, contact us at huasen520520@outlook.com.

本ポリシーは FamilyNAS の iOS / Android クライアントに適用されます。FamilyNAS は公共クラウドストレージを提供しません。写真、動画、ファイルは、ユーザー自身が設置・管理する FamilyNAS サーバーに保存されます。

1. アクセスするデータ

データ目的端末外へ送信されるか
写真、動画、Live Photosユーザーが選択したバックアップ、閲覧、プレビュー、共有アルバムユーザー自身の FamilyNAS サーバーにのみ送信
端末名、OS バージョン、アプリバージョンユーザーのサーバー上で端末を識別・管理するためユーザー自身の FamilyNAS サーバーにのみ送信

2. データの保存場所

写真、動画、共有アルバムの内容、ファイルメタデータは、ユーザー自身の NAS、Mac、Windows PC、または家庭内サーバーに保存されます。FamilyNAS クライアントは、ユーザーのメディアファイルを当社の公共サーバーへアップロードしません。

3. 共有と共有アルバム

FamilyNAS は、子どもの成長、旅行、誕生日、行事、家族の大切な瞬間を共有アルバムで記録・共有できます。共有内容は、ユーザー自身のサーバーと許可された家族メンバーのみがアクセスできます。

4. セキュリティ

  • クライアントとサーバーは HTTPS/TLS で通信します。
  • 証明書フィンガープリント検証により、中間者攻撃のリスクを低減します。
  • 家族メンバーは個別アカウントを使用します。ファイルは既定で非公開で、ユーザーが選択した内容のみ共有アルバムに表示されます。

5. 当社が行わないこと

  • 個人情報を販売、貸与、取引しません。
  • 広告 SDK を使用しません。
  • アプリや Web サイトをまたいだ追跡を行いません。
  • ユーザーの写真、動画、家族アルバムの内容を当社サーバーに保存しません。

6. ユーザーの管理

ユーザーはシステム設定で写真、カメラ、ローカルネットワーク、通知の権限を取り消せます。また、FamilyNAS サーバー管理ページからファイル削除、端末無効化、アカウント削除を行えます。クライアントをアンインストールすると、ローカルキャッシュとログイントークンが削除されます。

7. 子どものプライバシー

FamilyNAS は家族が子どもの成長記録を保存・共有するために利用できますが、当社が意図的に子どもの個人情報を収集することはありません。関連するメディアファイルは、ユーザー自身のプライベートサーバーに保存されます。

8. お問い合わせ

プライバシーに関するお問い合わせは huasen520520@outlook.com までご連絡ください。
