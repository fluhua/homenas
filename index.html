常见问题

你可能想问的

FamilyNAS和百度网盘 / iCloud 有什么区别？
FamilyNAS的数据完全存储在你自己的硬件上——电脑、树莓派或 NAS 设备。没有月费订阅、没有隐私泄露风险、没有容量限制（只取决于你的硬盘大小）。照片文件不需要经过第三方服务器。
服务器对硬件有什么要求？
几乎没有硬性要求。Go 编写的服务端非常轻量——空闲时内存占用不到 20MB。你可以跑在日用电脑（macOS/Windows）、闲置旧电脑、树莓派或任何 Linux 设备上。存储空间取决于你的硬盘大小。
需要一直开着电脑吗？
是的，服务端需要持续运行才能接收备份。但资源占用极低，不影响日常使用。建议让电脑保持开机（合盖不关机），或者用一台闲置设备专门运行。
外面能用吗？如何从外网访问？
FamilyNAS默认在局域网内工作。如果需要从外网访问，建议通过 Tailscale / ZeroTier 等虚拟组网工具，或者配置反向代理（Nginx / Caddy）并开启 HTTPS。不推荐直接将端口暴露到公网。
数据安全吗？
所有通信基于 TLS 加密，客户端通过证书指纹固定防止中间人攻击。数据存储在你的本地网络内，攻击面远小于公有云。建议保持服务端更新、使用强密码、不在公网暴露端口。
多台 Android 手机可以同时备份吗？
可以。每台手机用不同账户绑定后，各自的文件和备份完全隔离——即使连接同一个服务端，用户也看不到彼此的文件（管理员除外）。共享相册是跨用户的协作空间。
支持哪些类型的文件？
支持照片和视频备份。服务端自动生成缩略图，Web UI 和 App 均可直接预览和播放。
换手机了怎么办？
在新手机上安装 App，用同一账户的注册码重新绑定即可。已备份的文件不受影响。如需注销旧设备，可在 Web 管理面板的「设备」页撤销旧设备的 Token。
如何更新服务端版本？
下载新版本安装包覆盖安装即可。v1 到 v2 的升级会自动将原有数据迁移到 SQLite 数据库，无需手动操作。原始 JSON 文件会保留，可作为回滚备份。
目前支持哪些平台？
服务端支持 macOS 12+ 和 Windows 10+。客户端支持 Android 7.0+。iOS 客户端正在开发中。
FamilyNAS是免费的吗？
是的，完全免费。没有订阅费、没有容量限制、没有功能锁。
