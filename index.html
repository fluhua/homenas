使用指引

从零部署到全家使用

1

下载并启动服务端

解压 nas-server-*.tar.gz，在终端运行 ./nas-server --config ./config.yaml。首次启动会自动创建配置文件，编辑 admin_password 设定管理员密码。

2

登录 Web 管理面板

浏览器打开 https://<服务器IP>:8443，用管理员账号登录。可在「用户」页创建家庭成员账户，在「设备」页生成注册码。

3

手机安装 App 并配对

从 App Store / Google Play（或直接下载 APK）安装 HomeNAS。打开 App 会自动扫描局域网中的服务器，点击连接后输入注册码即可完成绑定。

4

开启自动备份

在 App「备份」标签页中开启自动备份。连接 Wi-Fi 时手机相册将自动上传。你可以设置在 Wi-Fi 下才备份、仅充电时备份等偏好。

5

浏览、分享与协作

在「云端文件」标签查看已备份内容。创建共享相册与家人协作整理，通过「分享链接」安全地分享给外部访客。

常见问题

你可能想问的

HomeNAS 和百度网盘 / iCloud 有什么区别？
HomeNAS 的数据完全存储在你自己的硬件上——电脑、树莓派或 NAS 设备。没有月费订阅、没有隐私泄露风险、没有容量限制（只取决于你的硬盘大小）。照片文件不需要经过第三方服务器。
需要一直开着电脑吗？
是的，服务端需要持续运行才能接收备份。但 Go 编写的服务端十分轻量——空闲时内存占用不到 20MB。你可以把它跑在闲置的旧电脑、树莓派、软路由或任何 Linux 设备上。
外面能用吗？如何从外网访问？
HomeNAS 默认在局域网内工作。如果你需要从外网访问，建议通过 Tailscale / ZeroTier 等虚拟组网工具，或者配置反向代理（Nginx / Caddy）并开启 HTTPS。我们不推荐直接将端口暴露到公网。
数据安全吗？被攻击了怎么办？
所有通信基于 TLS 加密，客户端通过证书指纹固定防止中间人攻击。由于数据存储在你的本地网络内，攻击面远小于公有云。建议：保持服务端更新、使用强密码、不在公网暴露端口。
多台手机可以同时备份吗？文件会混在一起吗？
可以。每台手机用不同账户绑定后，各自的文件和备份完全隔离——即使连接同一个服务端，用户也看不到彼此的文件（管理员除外）。共享相册是跨用户的协作空间。
支持 Live Photo 和视频备份吗？
iOS 端完整支持 Live Photo（照片 + 视频分别上传并关联）；Android 端支持照片和视频。服务端自动生成缩略图，Web UI 和 App 均可直接预览和播放。
服务端支持哪些平台？
服务端提供 macOS (amd64/arm64) 和 Linux (amd64/arm64) 的预编译二进制。同时提供 macOS 原生桌面应用（.dmg）。Windows 用户可通过 WSL2 运行 Linux 版本。
HomeNAS 是免费的吗？开源吗？
是的，完全免费。项目源代码开放，你可以自行审查、修改和重新分发。
